# Keycloak : les notions de base

## Ce que Keycloak apporte
- SSO (OIDC/SAML) pour des applications web et APIs
- Gestion utilisateurs / groupes / rôles
- MFA (OTP, WebAuthn/FIDO2)
- Fédération (LDAP/AD)
- Admin API / automation

## Objets clés (à maîtriser)
1. Realm
2. Client (OIDC/SAML)
3. User + Groupes + Rôles
4. Mappers (claims)
5. Flows (authent, registration, reset)
